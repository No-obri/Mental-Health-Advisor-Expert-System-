Mental Health Advisor Expert System
This is a Python-based expert system that evaluates user-reported symptoms to suggest possible mental health conditions such as depression, anxiety, burnout, or insomnia. It uses rule-based AI with forward chaining to provide diagnosis suggestions and self-care advice.

Features
Terminal and GUI modes

Rule-based inference engine

Confidence scoring for each condition

Advice and suggestions for matched conditions

Diagnosis report generation

Graph visualization in GUI
